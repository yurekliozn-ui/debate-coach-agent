# Debate Coach Agent

An AI powered agent that researches any debate topic, structures 
arguments for both sides, scores their strength, and outputs a 
ready-to-use debate preparation sheet.

## System Overview
The Debate Coach Agent accepts a debate topic from the user, 
researches it using live web search, and produces a structured 
document containing:
- A neutral topic summary
- 3–5 PRO arguments with source references
- 3–5 CON arguments with source references
- Strength scores (1–10) for each argument
- Key counterpoints
- Recommended sources
